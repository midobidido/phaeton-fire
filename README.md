# Phaeton Fire

Official website for Phaeton Fire — the ska-punk-reggae-prog space opera family band and in-universe soundtrack to The Loopborn Saga by Mike Van Doorn / Outer Spaces Studios.

## Local dev

Open `index.html` directly in a browser, or serve with any static file server:

```
npx serve .
```
